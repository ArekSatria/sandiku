# SANDISCAN Frontend

Frontend SANDISCAN dibangun menggunakan React.js dan Vite. Frontend berfungsi sebagai antarmuka pengguna untuk menganalisis kata sandi serta sebagai antarmuka admin untuk melihat dashboard statistik.

## Teknologi Frontend

- React.js
- Vite
- Axios
- React Router DOM
- CSS custom
